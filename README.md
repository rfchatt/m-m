## M&M Mobile Store

This project is a commercial management web application designed to help businesses efficiently track daily sales, dealer balances, credits, and cash flow. It provides an intuitive interface for entering and viewing daily financial data, including remaining balances for multiple dealers, sales totals, and pending credits. The application supports per-day data management, allowing users to review and update records for any selected date, and ensures all information is securely stored locally for easy access and backup. With features like margin calculation, product and credit tracking, and PDF export, this tool streamlines commercial operations and daily reporting.
